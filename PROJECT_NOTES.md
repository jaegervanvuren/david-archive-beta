# D.A.V.I.D. Project Notes

## Brand
D.A.V.I.D. = Digital Archive of Vision, Imagination, and Design.

## Current Site Logic
The beta uses one main React page with modular data in `src/data/siteData.js`.

## Content Strategy
- Keep Salvador as the central IP portal.
- Keep the Learning Lab school-safe and professional.
- Keep A Nation of Priests as a separate documentary/journal wing.
- Keep Idol Smasher Pedagogy as the academic/professional framework.
- Use Development Vault for archives, copyright records, PDFs, and project timelines.

## Suggested Beta Milestone
Before adding advanced features, make the homepage beautiful and navigable, then add real content one portal at a time.
