# D.A.V.I.D. Beta Website

**D.A.V.I.D.**  
**Digital Archive of Vision, Imagination, and Design**

This is a Vite + React beta website package for an integrated archive containing:

- Salvador Universe
- D.A.V.I.D. Learning Lab
- A Nation of Priests
- Idol Smasher Pedagogy
- Development Vault

## How to run locally

Open the folder in Command Prompt or Terminal and run:

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal, usually:

```text
http://localhost:5173/
```

## How to edit

Main page:

```text
src/App.jsx
```

Main content data:

```text
src/data/siteData.js
```

Main styling:

```text
src/styles.css
```
